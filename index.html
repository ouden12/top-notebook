Javascript Fundamentals

Javascript code can be run on browser or outside the browser environment, for now we will cover only the first one. To add JS code to our document we can do so by using the 'script' tag and adding the code inside it, or by linking an external file with the attribute 'src' and the doc path inside the tag.


Variables

There is now a way to declare variables that wasn't used before. We can use 'let' anv 'const', before the only way of declaring variables was using 'var', there are some cases where var will behave in odd ways in comparison to using just 'let' and 'const'. To declare a variable we just need to use the keyword 'let' and the name of said variable. To add data to our variable we can use the operator '=', like this:

let doggo_name;
doggo_name = "Catalina";

We can be more concise and combine the declaration of the variable with the assignment in a single line 'let variable = value;', note that we can declare multiple variables in one single 'let' statement. Then we can proceed to use the variable's content by using it's name. Like this:

let doggo_name = "Catalina", doggo_happiness = 100;
alert(doggo_name);

There are a couple of limitations to the naming of variables in JS. We can only use letters, digits or the symbols '$' and '_'. Also, the first character of the variable MUST NOT be a digit.
