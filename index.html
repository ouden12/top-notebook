Javascript Fundamentals

Javascript code can be run on browser or outside the browser environment, for now we will cover only the first one. To add JS code to our document we can do so by using the 'script' tag and adding the code inside it, or by linking an external file with the attribute 'src' and the doc path inside the tag.


Variables

There is now a way to declare variables that wasn't used before. We can use 'let' anv 'const', before the only way of declaring variables was using 'var', there are some cases where var will behave in odd ways in comparison to using just 'let' and 'const'. To declare a variable we just need to use the keyword 'let' and the name of said variable. To add data to our variable we can use the operator '=', like this:

let doggo_name;
doggo_name = "Catalina";

We can be more concise and combine the declaration of the variable with the assignment in a single line 'let variable = value;', note that we can declare multiple variables in one single 'let' statement. Then we can proceed to use the variable's content by using it's name. Like this:

let doggo_name = "Catalina", doggo_happiness = 100;

// Then we proceed to show the information on screen with the alert() function,
// but implementing it within another function when the user clicks on a button.

let doggo_Button = document.getElementById("printDoggo");

doggo_Button.addEventListener("click", printDoggo);

function printDoggo ()
{ alert(doggo_name+" is "+doggo_happiness+" levels of happy."); }

There are a couple of limitations to the naming of variables in JS. We can only use letters, digits or the symbols '$' and '_'. Also, the first character of the variable MUST NOT be a digit. Also the variables are case sensitive so 'Number' and 'number' are two separate variables.


Numbers

I don't remember right now all the theory that I read regarding numbers, I know, I should have write this down before. Either way. The numbers, and pretty much all variables in JS are declared the same way as mentioned before. You can use 'let' for them.

We can use basic operations like addition, subtraction, division and multiplication. Also in JS the order of operations follow the same rules as basic math, this means that the '*' and '/' are executed before '+' and '-'. In case there are two operators of the same level then it goes from left to right.

There are also a couple of ienteresting things related to math in JS. For example we can use the "-" or "+" operator on an operand (the number being "operated(?"), so we can transform a number from positive to negative or the other war around.

Let's see an example of all this:

let numberA, numberB, result;
numberA = 200-100;
numberB = 250/5;
result = -result;
result = numberA*numberB; // This would be -5000. Check the console if you don't believe me ;)

Strings

Then again this isn't too complicated either. We actually use an example for this first section. The interesting things about strings is how much we can do with them and is a lot. At the moment I only studied basic possible operations. Like these ones:

let cadena;
cadena = "Lorem ipsum dolor sit amet consectetur adipisicing elit. Beatae eaque excepturi similique maxime!";

// Extract a part of a string
// It can be done with three methods: `slice(start, end)`, `substring(start, end)` and `substr(start, end)`
// You can check which one adapts better to your specific case
let fragmento;
fragmento = cadena.slice(6, 20);
console.log("Mi fragmento es: "+fragmento); // This would print "Mi fragmento es: ipsum dolor sit amet"

// Get a string length
// We use the property `.length`
let largo;
largo = cadena.length();
console.log("The string has "+largo+" characters."); // This would print: "The string has 97 characters."

// Replacing content of a String
// We use the method `.replace(oldText, newText)`
let frase = "Los hobbits comen bien. A los hobbits les gusta la naturaleza";
let fraseModificada = frase.replace('hobbits','elfos');
console.log(frase);
console.log(fraseModificada); // New text but only one instance modified

// We can use a regular expression (/Lorem/) with the /g flag set to replace multiple matches
// Also, we can use the /i flag to find matches without the same capitalization
fraseModificada = replace(/hobbits/gi, "elfos");
console.log(fraseModificada); // This would print: "Los elfos comen bien. A los elfos les gusta la naturaleza"
